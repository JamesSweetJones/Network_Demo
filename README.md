# Network Demo

This html file is the output graph neural network mapping 328 nodes representing drugs, targets and related proteins with 1490 edges drawn between them based on their appearence in 1000 patents describing therapeutics in a muscular disease. This network is the output of a proprietary data science pipeline from Revena and so only outputs are given as a demo. This was graph was constructed in RustworkX and rendered using PyVis.

To view the graph, download the html file and open in your browser. The graph will render as a live searchable network.
